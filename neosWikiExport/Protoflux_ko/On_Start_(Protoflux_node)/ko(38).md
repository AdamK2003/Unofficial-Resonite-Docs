**On Start** 노드는 노드가 초기화되었을 때 임펄스를 발생시킵니다. 각
유저마다 별도로 임펄스가 발생되며, `Only Host`가 참일 경우, 호스트
유저에게만 발생합니다.

## 사용방법

이 노드가 활성화되는 조건은

-   인벤토리에서 생성될 때
-   월드 시작시
-   새 유저가 월드에 들어올 때
-   복제될 때
-   되돌리기 / 다시 실행 등을 통해 다시 로딩될 때
