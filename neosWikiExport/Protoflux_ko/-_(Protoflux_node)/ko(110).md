**-** 노드는 입력 A에서 입력 B의 값을 뺍니다.

## 사용방법

이 노드는 기본적으로 float 자료형을 사용하지만 적절한 와이어가 연결된
경우 다양한 다른 숫자 데이터 유형을 처리하도록 형변환(cast)할 수
있습니다. 입력이 3개 이상이 있는 경우 입력 이름은 Operands.\[0\],
Operands.\[1\]...로 변경됩니다. 이 경우 모든 입력은 Operands.\[0\]에서
뺍니다.
