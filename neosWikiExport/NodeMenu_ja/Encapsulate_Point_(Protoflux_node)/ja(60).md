**Encapsulate Point** ノードは、`Bounds` の BoundingBox と float3 の
`Point` を含む BoundingBox の計算結果を返します。

## 使用方法

`Bounds` と `OtherBounds`
のデフォルト値は、中心が\[0,0,0\]、サイズが\[0,0,0\]の BoundingBox
で、`Point` のデフォルト値は\[0,0,0\]です。
