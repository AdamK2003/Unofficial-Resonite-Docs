**Twitch Chat Message Event**ノード

## 使用方法

On
Message出力で生成されたインパルスは、接続されているTwitchInterfaceコンポーネントのTargetUserフィールドで参照されるユーザーが所有します。このフィールドが空白（Null）である場合、または参照されているユーザーがワールドに存在しない場合、デフォルトでホストユーザーとなります。
