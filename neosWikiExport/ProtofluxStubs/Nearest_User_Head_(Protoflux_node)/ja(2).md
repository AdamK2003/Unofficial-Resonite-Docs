## 使用方法

参照されたスロットに最も近いアバターのヘッドスロットを取得します。参照スロットに何もない場合、ノードは自分自身を参照として使用します。
手の代わりに頭を探す以外は [Nearest User Hand (Protoflux
node)](Nearest_User_Hand_(Protoflux_node) "wikilink")
に非常によく似た動作をします。
